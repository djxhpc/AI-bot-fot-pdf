#### 環境建立

環境 建立
全新的要下
sudo apt update
安裝套件
sudo apt install python3-venv 
建立環境
python3 -m venv penguinchuan

環境 啟動
source penguinchuan/bin/activate

離開環境

deactivate

Window 終端
Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope Process
若顯示停用令碼就使用

pip install virtualenv
Python  -m venv penguinchuan 
penguinchuan\Scripts\activate

Windows
py -3.10 -m venv penguinchu

.\penguinchu\Scripts\activate
